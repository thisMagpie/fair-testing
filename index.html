fair-testing

Clone

git clone https://github.com/thisMagpie/fair-testing
cd fair-testing
git fetch origin
git checkout master

To run java

The entries are still hard coded for now so all that is needed is to compile

cd fair-testing/java
javac *.java

Run using

java RunTest

To compile the latex standalone graphs

cd fair-testing/java/output
pdflatex output-datafile.tex

The first set reside in the java/output/one folder for the first test which is a single subject 100 times assuming a 3% error on their score (arbitrary estimate).

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
